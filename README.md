# Premium Developer Portfolio

A professional, high-performance developer portfolio built with React, Vite, and Tailwind CSS. This project is designed to showcase skills, projects, and services with a premium aesthetic and smooth user experience.

## 🚀 Features

- **Modern UI/UX**: Clean, professional design with a focus on typography and spacing.
- **Responsive Design**: Fully mobile-responsive layout.
- **Smooth Animations**: Powered by Framer Motion for a fluid feel.
- **Interactive Sections**:
  - Hero Section with dynamic text.
  - Interactive Project Gallery.
  - Services & Pricing overview.
  - Testimonials slider.
  - Contact Form integration.
- **Micro-interactions**: Hover effects, scroll progress bars, and back-to-top functionality.

## 🛠️ Built With

- **Framework**: React 18
- **Bundler**: Vite
- **Styling**: Tailwind CSS
- **Components**: shadcn/ui (Radix UI)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Icons**: Recharts for data visualization

## 📦 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or bun

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```
